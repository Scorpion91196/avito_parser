***Avito Parser***

    - Python 3.6
    - BS4
    - PyQt5
    - ...

*Установка*

- pip install -r req.txt

**Описание**

- Используется многопоточность
- Выходной файл CSV можно прочитать через Exel и составить красивую таблицу с данными
- Парсит:
    - Название
    - Ссылку на объявление
    - Цену
    - Описание
